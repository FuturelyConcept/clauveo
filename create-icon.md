# Creating App Icons

## Quick Solution: Use Any PNG Image

1. Find any PNG image (32x32, 128x128 pixels)
2. Rename them to match the required names:
   - `32x32.png`
   - `128x128.png` 
   - `128x128@2x.png`
   - `icon.ico` (for Windows)
   - `icon.icns` (for macOS)

3. Place them in `src-tauri/icons/`
